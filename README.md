# Systems and Projects

## Seqrview Examination Monitoring Platform

Working on backend infrastructure and operational monitoring for seQRView, a platform used in large scale examination environments to verify authorized staff and prevent impersonation using offline QR based identity verification.

The system runs as a live production platform handling more than ten thousand activity events during examination operations. Reliability and system stability are critical in these deployments.

My role focuses on maintaining backend services, monitoring system behavior, debugging production issues, and improving infrastructure efficiency.

Key contributions

Managing and monitoring a live AWS hosted production server handling more than 10k operational activities
Debugging backend APIs, worker pipelines, and database operations during active deployments
Working with Redis based task queues and caching layers to stabilize background processing
Maintaining operational data pipelines across relational databases and MongoDB
Investigating discrepancies across system logs, APIs, and service workflows
Optimizing database usage and query patterns which helped reduce infrastructure costs
Supporting infrastructure used during large scale examination deployments

## AI Auditor for Seqrview

Built an internal auditing system designed to monitor the Seqrview platform and detect discrepancies across backend services.

The system analyzes logs, API responses, and service states using Gemini based reasoning workflows. It identifies inconsistencies in the system and suggests or performs corrective actions to restore expected behavior.

The long term goal of this system is to move toward self monitoring infrastructure where routine operational issues can be detected and resolved automatically.

Capabilities

Automated analysis of backend logs and service behavior
Detection of inconsistencies across APIs and service workflows
AI assisted debugging and correction suggestions
Early exploration of self healing infrastructure workflows

Stack

Python
Gemini API
System monitoring pipelines

## Biometric Verification Platform

A fingerprint verification system built for authentication workflows across desktop and mobile devices.

The system preprocesses fingerprint images, detects thumb regions automatically, aligns fingerprint silhouettes, and performs verification through REST APIs.

Key work

CLAHE preprocessing for ridge enhancement
Automatic thumb detection using MediaPipe
Silhouette alignment for consistent fingerprint comparison
Integration of OpenCV.js and TensorFlow.js for preprocessing and analysis
Cross device testing and responsive UI support

Stack

Django
Django REST Framework
OpenCV.js
TensorFlow.js
MediaPipe
JavaScript
